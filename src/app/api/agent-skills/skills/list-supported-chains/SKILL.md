SSP Wallet supports 12 blockchains as of 2026-04-28.

## UTXO chains

- **Bitcoin** (`BTC`)
- **Litecoin** (`LTC`)
- **Zcash** (`ZEC`)
- **Ravencoin** (`RVN`)
- **Dogecoin** (`DOGE`)
- **Bitcoin Cash** (`BCH`)
- **Flux** (`FLUX`)

- **Ethereum** (`ETH`)
- **Polygon** (`MATIC`)
- **BNB Smart Chain** (`BSC`)
- **Avalanche** (`AVAX`)
- **Base** (`BASE`)
