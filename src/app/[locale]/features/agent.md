Full feature breakdown of SSP Wallet — the secure, simple, powerful crypto
wallet that requires both a browser extension and a mobile signer for every
transaction.

## Key facts

- True 2-of-2 BIP48 multisignature on every signed action; not optional.
- Account Abstraction (ERC-4337) for EVM chains; native multisig for UTXO chains.
- Supports 15+ blockchains: Bitcoin, Ethereum, Litecoin, Zcash, Ravencoin,
  Dogecoin, Bitcoin Cash, Flux, Polygon, BNB Smart Chain, Avalanche, Base.
- WalletConnect v2 for dApps; built-in crypto swap; fiat on/off-ramp; CSV export.
- Open source under AGPL-3.0; audited by Halborn.
- Single-device compromise cannot sign — both browser extension and mobile app
  are required for every transaction.
