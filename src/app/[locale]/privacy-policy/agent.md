Privacy Policy covering the entire SSP Wallet ecosystem — websites
(sspwallet.io / sspwallet.com), the SSP Wallet browser extension, the SSP
Key mobile app, and the SSP Relay service. Issued by InFlux Technologies
Limited (UK).

## What this page covers

- Privacy-by-design principles: no private-key access, local-only encrypted
  storage, minimal data collection, consent-based analytics.
- Explicit list of what is NOT collected: keys, seed phrases, transaction
  history, balances, PII, browsing/behavioral analytics, location, IP for
  profiling.
- What IS collected: short-lived relay sync/transaction data (15-minute
  retention), Firebase push tokens, device fingerprints (encryption only,
  not tracking), standard server logs, optional Google Analytics (only
  after explicit consent).
- Third-party integrations called out: Onramper (fiat on/off-ramp),
  WalletConnect v2 (dApp connections), Firebase Cloud Messaging.
- Encryption: AES-GCM with PBKDF2, BIP48 hierarchical derivation, HTTPS/WSS.
- Open source AGPL-3.0; audited by Halborn.
- Not intended for users under 18.
