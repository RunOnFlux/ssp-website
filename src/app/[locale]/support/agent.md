Support hub for SSP Wallet — FAQ across three categories, channel directory,
and a ticket form. Use this page first for self-service before opening a
ticket.

## Key facts

- 38+ FAQ entries grouped into Getting Started, Security & Privacy, and
  Technical Support. Custom useState accordion (not Radix) — one item open
  at a time.
- Direct support channels: support@sspwallet.io, ticketing system at
  https://support.runonflux.io, GitHub issues, Discord, Twitter.
- Ticket form posts to https://relay.ssp.runonflux.io/v1/ticket with fields
  email, type (Question / Incident / Problem / Feature Request / Bug Report),
  subject, description.
- Page emits FAQPage and BreadcrumbList JSON-LD; FAQ entries indexable as
  Question/Answer pairs.

## What you can do

- Browse the FAQ — open each item to read the answer.
- File a ticket via the form (relay.ssp.runonflux.io/v1/ticket) or directly
  at https://support.runonflux.io.
- Email support@sspwallet.io for direct help.
- Need installation help → /guide
- Other inquiries (partnership, media, business) → /contact
- Discord community: https://discord.gg/runonflux
