General contact page — partnerships, media, business development, and
generic inquiries. For wallet-specific technical questions go to /support
first.

## Key facts

- Contact methods featured: ticketing at https://support.runonflux.io,
  Discord (https://discord.gg/runonflux), GitHub Issues
  (https://github.com/RunOnFlux/ssp-wallet).
- Email: support@sspwallet.io.
- Contact form posts to https://relay.ssp.runonflux.io/v1/contact with
  fields name, email, and a composed message that includes subject,
  company (optional), inquiry type (general / support / partnership /
  security / media / business), and free-text body.
- Page emits BreadcrumbList JSON-LD.

## When to use this page vs /support

- Use /support for technical wallet questions, FAQ lookups, troubleshooting,
  and ticketed bug reports.
- Use /contact for partnership outreach, media/press requests, business
  development, security disclosures, and general non-technical inquiries.

## What you can do

- Pick a contact channel from the cards (tickets / Discord / GitHub).
- Send a message via the form (relay.ssp.runonflux.io/v1/contact).
- Email support@sspwallet.io directly.
- Need installation help → /guide
- Need wallet support / FAQ → /support
