SSP Enterprise is the multisig vault product for organisations — true
self-custody at team scale. No custodians, no MPC vendors holding key shards.

## Key facts

- M-of-N multisig vaults; threshold (2-of-3, 3-of-5, etc.) is configurable per
  vault.
- Two devices per signer — browser extension + mobile app required for every
  signature. A single compromised device cannot sign.
- Role-based access: Owner, Admin, Member, Viewer. Member changes are logged.
- Built-in transaction policy engine: spending limits, address whitelists,
  time-lock delays, admin approval thresholds, per-signer overrides, velocity
  tracking, emergency vault freeze — 14+ controls in a 3-tier hierarchy
  (organization → vault → signer).
- Multi-chain: Bitcoin, Ethereum, Litecoin, Dogecoin, Bitcoin Cash, Ravencoin,
  Zcash, Flux, Polygon, BSC, Avalanche, Base.
- Complete audit trail — every proposal, approval, rejection logged.
- In production with the Flux Foundation (Fusion bridge + treasury).
