Download hub for both halves of the SSP Wallet 2-of-2 multisignature system:
the browser extension (SSP Wallet) and the mobile companion app (SSP Key).
Both are required — the extension alone is not functional.

## Key facts

- Browser extension auto-detects Firefox vs Chromium and links to the correct
  store. Default landing CTA is the Chrome Web Store.
- Chrome Web Store:
  https://chromewebstore.google.com/detail/ssp-wallet/mgfbabcnedcejkfibpafadgkhmkifhbd
- Firefox Add-ons:
  https://addons.mozilla.org/en-US/firefox/addon/ssp-wallet
- Supported browsers: Chrome, Brave, Firefox, and any Chromium-based browser.
- SSP Key mobile app — the second-signature companion — is required to use
  SSP Wallet. Available for iOS and Android:
  - App Store: https://apps.apple.com/us/app/ssp-key/id6463717332
  - Google Play:
    https://play.google.com/store/apps/details?id=io.runonflux.sspkey
- Page emits BreadcrumbList JSON-LD.

## What you can do

- Click "Download Extension" to install on your current browser.
- Pick a browser tile (Chrome / Brave / Firefox) for direct store links.
- Install SSP Key on iOS or Android via the store badges.
